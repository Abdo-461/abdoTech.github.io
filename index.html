UniLink System

A new Java Challenge!

This was a small and fun school project where i was assigned to build a GUI application for university students who are interested in creating social events, selling or asking fellow unimates for help! The technology used for this application was Java 11 and JavaFX 14. This was first time developing an application using Java framework. I have learned several programming and developing techniques which have changed how i write code and build projects.

  • Date: March '20 -- June'20
  • Client: RMIT
  • Category: Java GUI application

Smart Grocer

Kitchen made easy and mobile!

This was my final year project in Swinburne University of Technololy - Sarawak Campus where my supervisor has established a company to offer innovative and easy solutions for the Malaysian State. I have lead this project and came up with the idea that we need to create a product rather than a solution. We have build a model cabinet that has different types of censors attached to different parts of the cabinet to measure contents and communicate values to a web and ios mobile application.

  • Date: June'17 - July'18
  • Client: Digital Futures
  • Category: Products solution
